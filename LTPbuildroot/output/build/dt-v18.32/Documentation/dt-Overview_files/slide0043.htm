Program Features
nData buffers are page aligned by default
ualign=value to misalign (e.g. align=rotate)
nPad bytes are allocated for data buffers
uInitialized with inverted pattern (1st 4 bytes)
uOn reads, verified after normal data
nCapacity is set via seek/reads or IOCTL
ucapacity=value – sets user capacity
nData verification is enabled by default
