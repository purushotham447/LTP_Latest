KeepAlive Messages
nOriginally added for HP’s Hazard to avoid PTO’s.
nEmits user defined message during long test runs.
nAllows you to roll your own pass/total messages.
nalarm=time controls frequency of keepalive msgs.
nkeepalive=string - defines the runtime message.
npkeepalive=string - defines per pass message.
ntkeepalive=string - defines the totals message.
nkeepalivet=time - keepalive message frequency.
nUsed when brief statistics are enabled (stats=brief):
uPer Pass Default:
«%d Stats: mode %i, blocks %l, %m Mbytes, pass %p/%P, elapsed %t
n      Totals Default:
«%d Totals: %L blocks, %M Mbytes, errors %e/%E, passes %p/%P, elapsed %T
