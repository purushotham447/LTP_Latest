Time Input Format
n Time Input:
n    d = days (86400 seconds),      h = hours (3600 seconds)
n    m = minutes (60 seconds),      s = seconds (the default)
n
n    Arithmetic characters are permitted, and implicit addition is performed on strings of the form '1d5h10m30s'.
n
nExample:
n # dt ... alarm=10s noprogt=1m runtime=24h30m15s
