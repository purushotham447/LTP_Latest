Data Test Program (dt)
49
DT Examples
nFor file system I/O tests:
u# dt of=/var/tmp/dt.data min=b max=256k incr=var oflags=trunc limit=1g \
u   pattern=iot iotype=random procs=10
u OR
u# dt of=/var/tmp/dt.data min=1 max=256k incr=var oflags=trunc limit=1g \
u   enable=lbdata files=10 sdirs=3 depth=10 procs=2
uAlternate between buffered I/O and direct I/O via flags=direct
nFor raw I/O disk tests:
u# dt of=/dev/rhdisk7 min=b max=256k incr=var pattern=iot slices=10 aios=4
nFor performance disk tests:
u# dt of=/var/tmp/dt.data bs=64k limit=1g disable=compare,fsync
u OR
u# dt of=/dev/rhdisk7 bs=64k aios=32 disable=compare
nIts advisable to alternate between sequential and random I/O, and when doing sequential, alternate between forward and reverse directions.
nSee Test Recommendations and Use Cases for more details.
