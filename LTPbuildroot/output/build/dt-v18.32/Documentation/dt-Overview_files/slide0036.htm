Trigger Script Arguments
nstring dname op dsize offset position lba errno noprogt
ustring = executable program or script
udname = The device or file name
uop = The operation in error
udsize = The device size
uoffset = The 64-bit file offset
uposition = Position in block of data corruption (DC)
ulba = The logical block address (real or relative LBA)
uerrno = The error number
unoprogt = The no-progress time value (in seconds)
