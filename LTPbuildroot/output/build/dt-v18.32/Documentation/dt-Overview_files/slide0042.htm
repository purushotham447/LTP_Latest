Data Test Program (dt)
11
Controlling Execution Time
nNormally each pass is controlled by the capacity or one of these options:
ulimit=value – Specifies the data limit.
urecords=value – Specifies record count.
ufor sequential I/O, stop at EOF/EOM.
ufor random I/O, loop until limit reached.
nruntime=time – Runtime in seconds.
uCareful: Short runtimes may prevent read/verify pass depending on limits.
npasses=value – Specifies pass limit.
