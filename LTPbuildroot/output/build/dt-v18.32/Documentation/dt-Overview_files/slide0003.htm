Data Test Program (dt)
6
General Command Format
nTo write then read/verify data:
u# dt of=filename bs=value limit=value …
nThe read/verify previously written data:
u# dt if=filename bs=value limit=value …
nTo write without doing read/verify pass:
u# dt of=filename bs=value disable=verify …
nTo read without doing data comparison:
u# dt if=filename bs=value disable=compare …
nTo specify the device type and device size:
u# dt of=filename dtype=type dsize=value …
nNote: filename is a valid path for your OS.
